== Description ==
 
Real time monitoring for Web Agency and Freelance.
Inspector allows you to find out application drawback before they affect your users. It send you
real-time alerts with complete diagnosis if something goes wrong in your website and
keep your Wordpress under monitoring 24/7 so you are free to focus on grow your
business avoiding tedious manual checks that your website is working properly.

Stop receiving report directly from users. Before a user spend time to send you
a report that something doesn't work in your website 10 others simply
left their navigation immediately.

Sometimes could be a slow plugin or a page with broken address. You will be able
to know what's impacting your customer journey and anticipate their needs in real
time and 24/7.

## How it works
Inspector collect execution information when a user navigate your Wordpress website
and send this data to your Inspector account to be analyzed.
It do this job silently because it start to transfer data after your website
was rendered in the user's browser.

It is simple, you just need to install this plugin. And safe because it is completely
transparent from your users experience.

## Look your Wordpress from another point of view
Now in just few clicks you can understand how long your theme take to load,
Witch widget is slowing down navigation, etc.

You will be clear visibility of how Wordpress components works
and which of them are helping you create a great experience
for your users and which are hindering you.

== Installation ==
1. Unpack the `download-package`.
2. Upload the file to the `/wp-content/plugins/` directory.
3. Activate the plugin through the `Plugins` menu in WordPress.
4. Insert your project api key.
5. Done and Ready.
